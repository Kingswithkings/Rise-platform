# Engineering Onboarding

## Prerequisites

- Git
- Docker Desktop with Docker Compose v2
- A GitHub account with repository access

## Start locally

```bash
git clone https://github.com/Kingswithkings/Rise-platform.git
cd Rise-platform
docker compose up --build
```

Verify the web application at `http://localhost:3000` and Swagger at
`http://localhost:8000/docs`. Read the architecture, development process, coding
standards, and testing strategy before making a change.
