# ADR-004 — Use Next.js

## Context

The web product requires React, TypeScript, server rendering, and a mature routing and
build system.

## Decision

Use Next.js for the web frontend.

## Alternatives considered

Client-only React, Remix, and non-React web frameworks.

## Consequences

The App Router owns page composition and the frontend is deployed as a separate container.

## Date

2026-07-01

## Status

Accepted
