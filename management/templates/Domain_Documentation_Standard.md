# Domain Documentation Standard

Every domain must include:

```text
backend/app/<domain>/
├── README.md
├── models.py
├── schemas.py
├── repository.py
├── service.py
├── router.py
├── validators.py
├── permissions.py
├── events.py
├── tests/
└── docs/
```

## Sprint Documentation Standard

Every sprint must include:

- Executive Summary
- Sprint Goals
- User Stories
- Architecture
- Database Changes
- API Documentation
- Implementation Notes
- Test Report
- Security Review
- Release Notes
- Retrospective
- Demo Checklist
