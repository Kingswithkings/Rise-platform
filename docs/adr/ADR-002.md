# ADR-002 — Use Docker

## Context

Development and deployment require consistent application and dependency environments.

## Decision

Package the frontend and backend with Docker and orchestrate local dependencies with
Docker Compose.

## Alternatives considered

Host-managed runtimes and virtual-machine development environments.

## Consequences

Docker becomes an onboarding prerequisite, while runtime parity and repeatability improve.

## Date

2026-07-01

## Status

Accepted
