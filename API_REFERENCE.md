# RISE API Reference

## API Documentation

FastAPI generates Swagger documentation automatically.

Local development:

```text
http://localhost:8000/docs
```

## Authentication

Protected endpoints use JWT Bearer authentication.

Login endpoint:

```text
POST /api/v1/auth/login
```
