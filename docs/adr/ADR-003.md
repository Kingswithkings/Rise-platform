# ADR-003 — Use FastAPI

## Context

The API requires typed Python contracts, asynchronous integrations, and automatic OpenAPI
documentation.

## Decision

Use FastAPI for the backend API.

## Alternatives considered

Django REST Framework, Flask, and Node.js frameworks.

## Consequences

Pydantic schemas define transport contracts and Swagger is generated from application
code.

## Date

2026-07-01

## Status

Accepted
