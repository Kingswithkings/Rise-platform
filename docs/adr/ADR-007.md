# ADR-007 — 1stKings Trust Branding

## Title

Use 1stKings Trust as the public product name

## Status

Accepted

## Context

The verification layer must develop recognizable trust beyond a generic marketplace
rating.

## Decision

Keep the internal module at `app/trust/`. Use the full 1stKings Trust brand in every
user-facing label, including 1stKings Trust Score, ID, Badge, Certificate, and
“Verified by 1stKings Trust.”

## Alternatives

- Generic Trust branding
- RISE Trust branding

## Consequences

The trust capability can develop independent recognition and later be extracted as a
shared service without renaming the customer-facing product.
