# Private Alpha Scope

The first private alpha should validate the core marketplace with real users before expanding into payments, shipping, AI recommendations, and advanced analytics.

## Buyers

- Register and log in
- Browse by region, country, and city
- Browse categories
- Search stores, products, and services
- View seller profiles
- Contact sellers

## Sellers

- Create a seller profile
- Create and manage stores
- Add products
- Add services
- Manage store information
- View basic dashboard

## Admin

- Manage users
- Manage categories
- Verify sellers
- Verify stores
- Review trust reports
- Moderate content
