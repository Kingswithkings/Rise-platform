# ADR-001 — Use PostgreSQL

## Context

RISE requires transactional consistency, relational integrity, and mature operational
tooling for marketplace and commerce data.

## Decision

Use PostgreSQL as the system of record.

## Alternatives considered

MySQL, document databases, and provider-specific databases.

## Consequences

Schemas evolve through Alembic, relational constraints enforce invariants, and production
operations require tested PostgreSQL backup and recovery.

## Date

2026-07-01

## Status

Accepted
